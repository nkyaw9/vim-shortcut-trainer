# vim-shortcut-trainer

## Description

Vim Shortcut Trainer is a beginner-friendly learning website for practicing Vim keyboard shortcuts. The goal of this project is to help users learn common Vim commands through examples, categories, and interactive practice.

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express
